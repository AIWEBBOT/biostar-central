Smart mode: sends an email when a user receives an answer or a comment on a question that they have posted. This is the recommended default for most users as it notifies them immediately when they get an answer.

Local messages: no emails. Local messages will be sent.

Local tracker: no emails. Generates a local message when there is activity on any thread that a user has contributed to with an answer or comment. This allows users to keep track of discussions they have contributed to.

Email tracker works like Local Tracker but also generates an email.

Mailing list: Generates an email for every post on the site. This allows you use the site as if it were a mailing list.

Leave Group: Removes the user from the group. When leaving a private group the user cannot rejoin unless added by the owner of the group.
